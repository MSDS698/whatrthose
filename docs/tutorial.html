Setup Tutorial¶

The following will walk through the setup, deployment, and execution of the application on a remote server.

Environment Setup¶

  1. Change working directory in terminal to git repository on local machine.
  2. Run conda env create -f environment.yml in terminal on local machine.

Deployment¶

  1. Specify in code/user_definition.py:
    ec2_address:EC2 Public DNS Address
    key_file:PEM Key file to access the EC2 instance
    git_user_id:Github User authorized to use repository
  2. Change working directory in terminal to git repository.
  3. Run python code/deploy.py in terminal.
